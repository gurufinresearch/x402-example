# Go implmenetation example for x402 server and client.


### Running the server
```
go run server/main.go
```
Documentation: [x402-server](https://github.com/gurufinresearch/x402/blob/main/go/SERVER.md)


### Running the client
```
go run client-cmd/main.go 
```
Documentation: [x402-client](https://github.com/gurufinresearch/x402/blob/main/go/CLIENT.md)

### Running the web example
```
go run client-web/main.go
```
